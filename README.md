# Charity Finder
A Charity Finder using Typescript. It uses Reactjs as frontend.

## 🎯 Features:  
- Charity list display
- Charity searching
- Add your favorite charity

## 🖥 Prerequisites
- Node.js (version 12.0 or above)
- npm (which comes with Node.js) or Yarn (version 1.22.0 or above)

## 🔧 Usage
### Build and Run
```
npm run dev
```
```
npm run build
```
